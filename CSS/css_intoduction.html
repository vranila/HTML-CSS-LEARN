CSS Introduction

CSS (Cascading Style Sheets) is the language used to style and design web pages. It describes how HTML elements should be displayed on screen, on paper, or in other media formats.

What is CSS?

  • CSS stands for Cascading Style Sheets.
  • It defines how HTML elements are displayed across devices.
  • It allows you to apply consistent design to multiple pages simultaneously.
  • External CSS files (.css) make global styling changes easy.

Why Use CSS?

CSS simplifies web design by controlling layout, color, font, and responsiveness. It helps deliver a consistent look across devices and allows one central place to modify styles.

CSS Example

    body {
    background-color: lightblue;
    }

    h1 {
    color: white;
    text-align: center;
    }

    p {
    font-family: verdana;
    font-size: 20px;
    }

Explanation:

  • background-color sets the background shade.
  • color defines text color.
  • text-align adjusts text alignment.
  • font-family changes text font style.
  • font-size sets text size.

CSS Syntax

CSS (Cascading Style Sheets) is written using a simple syntax that defines how HTML elements should appear on a webpage.

Structure of a CSS Rule

A CSS rule consists of two main parts:

  • Selector - targets the HTML element(s) you want to style.
  • Declaration block - contains one or more style declarations enclosed in curly braces { }.
    selector {
    property: value;
    property: value;
    }

Each declaration includes a CSS property name and a value, separated by a colon (:), and declarations are separated by semicolons (;).

Example

The following rule styles all <p> elements with red color and centered text:

p {
  color: red;
  text-align: center;
}

Example Explained

  • p - is the selector that points to all paragraph elements.
  • color - is a property that changes text color.
  • red - is the value assigned to the color property.
  • text-align - aligns text horizontally within its element.
  • center - is the value making the text centered.

Summary

  • A CSS rule = Selector + Declaration Block
  • Declaration Block = Property + Value pairs
  • Use ; to separate multiple declarations
  • Wrap declarations in { }
