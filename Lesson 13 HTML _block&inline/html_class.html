5. Summary Table

Feature Description Example
Assign a class Adds a name to an element for styling or scripting <p class="highlight">Text</p>
Multiple classes Separate names by spaces <div class="card shadow"></div>
Used in CSS To style elements with specific classes .highlight { color: blue; }
Used in JavaScript To manipulate elements dynamically element.classList.toggle('active')

6. Summary

  • class → Assigns one or more identifiers to an element
  • Shared usage → Multiple elements can share the same class
  • Used by → CSS (for styling) and JavaScript (for manipulation)
