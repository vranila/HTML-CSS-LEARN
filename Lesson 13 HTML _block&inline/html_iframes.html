HTML Iframes

Definition: An HTML <iframe> (inline frame) is used to display a web page inside another web page.

1. HTML Iframe Syntax

The <iframe> tag specifies an inline frame.

<iframe src="url" title="description"></iframe>

Tip: Always include the title attribute. It helps screen readers describe the iframe's content for accessibility.

5. Summary Table

Attribute / Property Description Example
src Specifies the URL of the page to embed <iframe src="page.html"></iframe>
title Describes the iframe content (for accessibility) <iframe title="Example Iframe">
height / width Defines the iframe's size <iframe height="200" width="300">
name Used to target links to load in the iframe <iframe name="myFrame">
style="border:none;" Removes the iframe border <iframe style="border:none;">

6. Chapter Summary

  • The <iframe> tag defines an inline frame.
  • src specifies the embedded page URL.
  • Always include a title for accessibility.
  • height and width define iframe size.
  • Use border:none; to remove borders.
  • name allows targeting links to the iframe.
