Chapter Summary

  • <dl> defines a description list.
  • <dt> defines a term or name in the list.
  • <dd> defines the description of that term.
  • These tags are ideal for definitions, FAQs, or key-value content.

HTML List Tag Reference

Tag Description
<ul>Defines an unordered (bulleted) list
<ol>Defines an ordered (numbered) list
<li>Defines a list item
<dl>Defines a description list
<dt>Defines a term or name in a description list
<dd>Describes the term or name in a description list
