HTML Character Entities

Character Entity Name Entity Number Description
< &lt; < Less-than sign
> &gt; > Greater-than sign
& &amp; & Ampersand
" &quot; " Double quotation mark
' &apos; ' Single quotation mark / Apostrophe
  &nbsp;   Non-breaking space
© &copy; © Copyright symbol
® &reg; ® Registered trademark symbol
€ &euro; € Euro symbol
£ &pound; £ Pound sterling symbol
¥ &yen; ¥ Yen symbol
¢ &cent; ¢ Cent symbol
™ &trade; ™ Trademark symbol

Notes

  • HTML entities start with & and end with ;.
  • They are used when a character is reserved or cannot be typed directly.
  • Example: To show <html>, use &lt;html&gt;.
  • Common entities include &nbsp; (space), &copy; (©), and &euro; (€).
