2. Rules for Using the id Attribute

  • Each id must be unique within an HTML document.
  • The id name is case sensitive.
  • The id must contain at least one character.
  • The id cannot start with a number or include spaces.

7. Chapter Summary

  • The id attribute assigns a unique identifier to an HTML element.
  • Each id must be unique within a document.
  • Used by CSS and JavaScript for targeting and manipulation.
  • Case sensitive and cannot start with a number.
  • Also used to create bookmarks on web pages.
  • Accessed in JavaScript using getElementById().
