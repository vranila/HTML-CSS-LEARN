HTML <div> Element

The <div> element acts as a container for structuring content on a web page. It’s block-level by default, meaning it occupies full width and starts on a new line.

Example: Basic <div> Usage

Code:

Welcome to our website <div>I am a separate block</div> Explore more below.

Result:

Welcome to our website
I am a separate block of content.
Explore more below.

Summary of Layout Methods

Method CSS Used Description
Float float: left; Classic method, requires clearing for layout control.
Inline-block display: inline-block; Simple alignment without floats.
Flexbox display: flex; Responsive layout control, easy vertical/horizontal centering.
Grid display: grid; Best for advanced layouts with rows and columns.
