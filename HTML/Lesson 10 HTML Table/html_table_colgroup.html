Key Notes

  • <colgroup> — Defines groups of columns to apply styles.
  • <col> — Defines individual column or column span styles.
  • span — Specifies how many columns are affected by a single <col>.
  • Legal CSS properties: width, visibility, background, border.
  • Place <colgroup> right after <caption> (if present) and before any rows.
