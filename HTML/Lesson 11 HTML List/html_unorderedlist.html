Chapter Summary

  • <ul> defines an unordered list.
  • <li> defines individual list items.
  • list-style-type controls bullet style (disc, circle, square, none).
  • Lists can be nested and combined with CSS for menus.
  • Use float:left; for horizontal navigation lists.
