HTML - The <head> Element

The HTML <head> element is a container for metadata — information about the document that is not displayed directly on the page. It sits between the <html> and <body> tags and defines critical information such as the page title, character encoding, styles, links, and scripts.

1. Structure of the <head> Element

The <head> section typically includes the following elements:

  • <title> - Defines the title shown in the browser tab.
  • <style> - Adds internal CSS styles.
  • <meta> - Specifies metadata like description, author, and viewport.
  • <link> - Links to external files such as stylesheets or icons.
  • <script> - Embeds or links JavaScript code.
  • <base> - Sets a base URL or default target for all links.

2. The <title> Element

The <title> element defines the document's title. It appears on the browser tab, in bookmarks, and is critical for SEO.

<title>MindCraft AI - Learn Smarter</title>

Key Benefits:

  • Displayed on browser tabs.
  • Used when adding a page to bookmarks.
  • Improves search engine ranking.

3. The <style> Element

The <style> element allows you to include internal CSS styling within the HTML document.

<style>
  body { background-color: #eef5ff; }
  h1 { color: #0a66c2; }
  p { font-size: 16px; color: #333; }
</style>

4. The <link> Element

The <link> tag connects your HTML document to external resources — most commonly external CSS stylesheets.

<link rel="stylesheet" href="styles.css">

It can also be used for favicons or preloading resources.

5. The <meta> Element

<meta> tags define metadata such as character encoding, page description, author, keywords, and viewport settings. They don’t render visible content but are essential for browser behavior and SEO.

Examples:

<meta charset="UTF-8">
<meta name="description" content="AI learning platform for future-ready skills.">
<meta name="keywords" content="AI, HTML, CSS, JavaScript, MindCraft">
<meta name="author" content="MindCraft AI">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

6. Setting the Viewport

The viewport controls how a web page is displayed on different devices. This meta tag ensures responsiveness across mobile and desktop screens.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Explanation:

  • width=device-width — Sets the page width equal to the device’s screen width.
  • initial-scale=1.0 — Sets the initial zoom level.

7. The <script> Element

The <script> tag allows you to include JavaScript, either directly within the HTML or from an external file.

<script>
  function showMessage() {
    document.getElementById("demo").innerHTML = "Hello, MindCraft!";
  }
</script>

You can also link external scripts:

<script src="main.js"></script>

8. The <base> Element

The <base> tag sets a base URL for all relative links in a document. It must appear only once within the <head> section.

<base href="https://mindcraft.ai/" target="_blank">

Now, all relative links will open from this base URL.

9. Example: Complete Head Section

<!DOCTYPE html>
<html>
<head>
  <title>MindCraft AI - Learn HTML</title>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="HTML tutorial from MindCraft AI">
  <link rel="stylesheet" href="style.css">
  <style>
    h1 { color: #0056b3; }
  </style>
  <script src="app.js"></script>
</head>
<body>
  <h1>Welcome to MindCraft AI</h1>
</body>
</html>

10. Chapter Summary

Tag Description
<head> Container for all metadata.
<title> Defines the document title (mandatory).
<style> Defines internal CSS rules.
<link> Links to external stylesheets or icons.
<meta> Defines metadata like description, author, and viewport.
<script> Includes client-side JavaScript.
<base> Sets the base URL or target for all relative links.
