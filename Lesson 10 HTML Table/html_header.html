Notes

  • <th> → Defines a header cell in a table.
  • <td> → Defines a standard data cell.
  • colspan → Merges multiple columns into one header cell.
  • rowspan → Merges multiple rows into one header cell.
  • <caption> → Adds a title for the entire table.
  • By default, table headers are bold and centered.
