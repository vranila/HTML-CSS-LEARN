Explanation of Table Tags

<table> Tag

The tag defines the start and end of a table. It acts as the container that holds all rows and cells.

<tr> Tag

<tr> stands for “table row”. Each <tr> defines a horizontal row inside the table.A table can have multiple rows to display different sets of data.
Example: Data1 Data2

<th> Tag

<th>stands for “table header”. It defines a header cell, which by default appears in bold and centered. Headers help identify the meaning of each column or row.
Example: Name Age

<td> Tag

<td> stands for “table data”. It defines the standard data cells inside a row. Each represents one piece of information.
Example: John 25

Summary

  • <table> – defines the table structure.
  • <tr> – defines each horizontal row.
  • <th> – defines header cells (bold + centered).
  • <td> – defines standard data cells.
  • border-collapse – merges border lines for a clean layout.
  • padding – adds spacing inside cells for readability.
