Notes

  • width — Defines the horizontal size of the table, column, or cell.
  • height — Defines the vertical size of a table row or cell.
  • Use percentage (%) for responsive layouts (relative to parent container).
  • Use pixels (px) for fixed dimensions.
  • Always use border-collapse: collapse; for clean borders.
